Prerequisite
-----------
Following packages are should be installed and on the ROS_PACKAGE_PATH.
- https://github.com/gt-ros-pkg/hrl-haptic-manip.git

Compilation
-----------
1. make 'build' folder on the plugin root folder
2. cmake ..
3. make -j4
4. sudo make install
